## Установка
---
Клонировать репозиторий и перейти в него в командной строке
```terminal
git clone https://github.com/user/GD_solevr.git
cd GD_solver 
```
Собрать бинарный файл
```terminal
mkdir build
cd build
cmake ..
make
```
